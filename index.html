TeamCity Companion - your mobile companion app for TeamCity

View on GitHub

Why a mobile app?

TeamCity is a great and easy to use Continuous Integration server, but as the number of projects it manages increases, reviewing build results on mobile devices becomes cumbersome. It requires a lot of zooming, waiting and generally is not very nice to use. TeamCity companion aims to help with this, making it very easy to review your build results while minimizing the HTTP traffic to your server.

Features

The core features of TeamCity Companion are as follows:

  • Configure multiple TeamCity servers
  • View and filter projects and builds
  • Flag build types and set up notifications for build failures
